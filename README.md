# 🏋️ Fitness Business Dashboard

## 📊 Project Overview
This project analyzes the business performance of a fitness center.  
The dashboard helps track revenue, expenses, membership growth, and member health metrics such as BMI, BMR, and TDEE.

---

## 🎯 Business Problems
- Track total clients and trainers
- Monitor revenue, expenses, and profit
- Track monthly performance
- Monitor membership status
- Calculate BMI, BMR, and TDEE for members

---

## 📁 Dataset Description
The dataset contains:
- Member information
- Membership tiers (Platinum, Gold, Silver)
- Trainer data
- Monthly revenue and expenses
- Membership start and expiry dates

---

## 📊 Dashboard Pages
1. Home Page
2. Business Overview
3. Member Details
4. Fitness Calculator

---

## 🛠 Tools Used
- Power BI
- Excel
- Data Modeling
- DAX
